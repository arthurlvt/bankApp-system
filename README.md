# Bank System

A simple and secure banking system with account management, currency conversion, and real-time exchange rate fetching.

> [!WARNING]
> This repo is currently under developement so if there are some issues, don't mind it !

---

## Features

- **Account Management**: Create, access, and delete accounts with secure password protection.
- **Money Transfer**: Transfer money between accounts.
- **Currency Conversion**: Convert amounts between different currencies (USD, EUR, GBP).
- **Real-Time Exchange Rates**: Fetch and display current exchange rates using an external API.
- **Secure Password Storage**: Passwords are hashed and salted for secure storage.

---

## Prerequisites

- A C compiler (e.g., `clang` or `gcc`)
- `libcurl` for HTTP requests
- `make` for building the project
